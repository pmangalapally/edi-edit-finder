# EDI Edit Finder

A reference tool for EDI X12 transaction codes, starting with the **TA1 Interchange Acknowledgment** edit codes.

---

## EDI X12 TA1 Edit Codes — Web App

`index.html` is a fully self-contained, single-page web application that lists all standard EDI X12 TA1 acknowledgment and error/note codes with their descriptions. No build tools, no dependencies, no server required.

### How to use

1. **Download or clone** this repository.
2. **Open `index.html`** in any modern web browser (Chrome, Firefox, Edge, Safari).
3. Use the **filter box** to search in real time by:
   - Edit code (e.g., `001`, `A`, `R`)
   - Keyword in the description (e.g., `sender`, `duplicate`, `date`)
   - Impact or category (e.g., `reject`, `error`)
4. Click **Clear** to reset the filter.

The counter at the top right shows how many codes match your current filter.

## What is a TA1 Acknowledgment?

In EDI X12, the **TA1 (Interchange Acknowledgment)** segment is used to report the status of processing an interchange envelope — the outermost wrapper defined by the **ISA** (Interchange Control Header) and **IEA** (Interchange Control Trailer) segments.

A TA1 is generated by the receiving trading partner or clearinghouse when an interchange-level issue is detected. It contains two key data elements:

| Segment Element | ID   | Purpose |
|----------------|------|---------|
| TA1-04 | I17 | **Interchange Acknowledgment Code** — overall status: `A` (Accepted), `E` (Accepted with Errors), or `R` (Rejected) |
| TA1-05 | I18 | **Interchange Note Code** — specific error code (`000`–`031`) describing the nature of the issue |

### TA1 Acknowledgment Codes (TA1-04)

| Code | Meaning |
|------|---------|
| `A`  | Interchange accepted with no errors |
| `E`  | Interchange accepted but errors were noted |
| `R`  | Interchange rejected because of errors |

### TA1 Error/Note Codes (TA1-05) — Quick Reference

| Code | Description |
|------|-------------|
| `000` | No error |
| `001` | Control number mismatch between header and trailer |
| `002` | Control standards identifier not supported |
| `003` | Controls version not supported |
| `004` | Invalid segment terminator |
| `005`–`009` | Invalid or unknown sender/receiver IDs or qualifiers |
| `010`–`013` | Invalid authorization or security information |
| `014`–`021` | Invalid date, time, version, control number, or other header values |
| `022` | Invalid control characters |
| `023` | Premature end-of-file |
| `024` | Invalid interchange content (e.g., invalid GS segment) |
| `025` | Duplicate interchange control number |
| `026`–`027` | Invalid data or component element separator |
| `028`–`030` | Invalid deferred delivery request values |
| `031` | Invalid grade of service code |

Open `index.html` for the full interactive reference with filtering.
